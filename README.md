# NullsWorld-webgame-ui

NullsWorld-webgame-ui for PC.

### Run

Recommend to use yarn, it's faster.

**0x0.Install yarn**

```sh
npm install yarn -g
```

**0x1.Install dependencies with yarn**

```sh
yarn i
```

**0x2.Run develop**

```sh
yarn run dev
```

**0x3.Build release**

```sh
yarn run build
```
